# Mon Grammaire

**Mon Grammaire** is an Android application designed to help users learn and improve their French grammar skills. It offers a variety of interactive features, lessons, and tools to make learning engaging and effective.

## Features

- **Grammar Lessons:** A comprehensive list of grammar lessons covering various aspects of the French language.
- **Interactive Quiz:** Test your knowledge with fun and challenging quizzes.
- **Conjugation Tool:** Easily search for and learn verb conjugations.
- **Alphabet Learning:** Basics for beginners to get started with the French alphabet.
- **Translation Tool:** Integrated translation support to help understand complex phrases.
- **Favorites:** Save your most important lessons for quick access.
- **Search:** Quickly find specific topics or verbs within the app.

## Project Structure

The project is organized into several packages:
- `com.example.mongrammaire.courslist`: Management of grammar lessons.
- `com.example.mongrammaire.Quiz`: The game engine and quiz logic.
- `com.example.mongrammaire.horisontal_cardv.categories.conjugaison`: Verb conjugation features.
- `com.example.mongrammaire.Alphabets`: Resources for learning the alphabet.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/hix/monGrammaire.git
   ```
2. Open the project in **Android Studio**.
3. Sync the project with Gradle files.
4. Run the app on an emulator or a physical Android device.
